# Bookeepr

Bookeepr is a full stack application that can be used to keep track of books you have read.

## Technologies used
- NodeJS
- ExpressJS
- Postgres SQL
- RestAPI


## Run

Clone this repository
```bash
git clone https://github.com/kmr-ankitt/Bookeepr
cd Bookeepr
```

Install all required packages
```bash
npm i
```

Start the server
```bash
npm app.js
```
